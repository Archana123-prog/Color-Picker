# 🎨 Color Picker App

A simple and interactive **Color Picker Web App** built using **HTML, CSS, and JavaScript**.
It allows users to generate random colors and copy their HEX codes instantly.

### Live Link:
https://archana123-prog.github.io/Color-Picker/

## 🚀 Features

* 🎨 Generate random colors
* 🔢 Display HEX color code
* 📋 Copy color code to clipboard
* 💻 Clean and responsive UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🎯 How It Works

* Click on **Generate Color** → A random color appears
* HEX code updates automatically
* Click **Copy Code** → HEX code copied to clipboard
